SUMMARY: WEATHER PLOTS
In Relation to Latitude

Purpose: To analyze the weather across various cities, 547 in total, to see the effects the latitudinal position of a city will have on the various weather categories. To accomplish this, a raw data file was provided- but in my Python-API repository, an example of weather analysis using api calls for OpenWeatherMap can be seen. This project will be updated when I have more time, to leverage that api call to pull in more recent data. At this time, only 06/13/2018 was used to create these analyses.


Tools: Matplotlib within Jupyter Notebooks was used to create some basic scatterplots with the independent variable being latitude (x), and the dependent variable being: Maximum Temperature, Humidity (%), Cloudiness (%), Wind Speed (mph).


Data and visualizations resulting are analyzed with brief descriptions on what is seen within the each scatter plot.
